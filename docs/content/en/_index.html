Get Skaffold
Skaffold handles the workflow for building, pushing and deploying your application, allowing you to focus on what matters most:writing code.
An open source project from Google.
{{< blocks/section color="dark" >}} {{% blocks/feature4 icon="fas fa-feather-alt" title="Lightweight" %}} Skaffold is client-side only. With no on-cluster component, there is no overhead or maintenance burden to your cluster. {{% /blocks/feature4 %}} {{% blocks/feature4 icon="fa-globe" title="Works Everywhere" %}} Skaffold is the easiest way to share your project with the world: `git clone`, then `skaffold run`. Additionally, you can use profiles, local user config, environment variables, and flags to easily incorporate differences across environments. {{% /blocks/feature4 %}} {{% blocks/feature4 icon="far fa-gem" title="Feature Rich" %}} Skaffold has many essential features for Kubernetes-native development, including policy-based image tagging, resource port-forwarding and logging, file syncing, and much more. {{% /blocks/feature4 %}} {{% blocks/feature4 icon="fas fa-cogs" title="Optimized Development" %}} Skaffold's inner loop is tight and highly optimized, giving you instant feedback while developing. {{% /blocks/feature4 %}} {{< /blocks/section >}} {{< blocks/section color="light" >}} {{% blocks/feature icon="fa-download" title="Install Skaffold" %}} [Get Skaffold on Linux, macOS, and Windows.](/docs/getting-started/#installing-skaffold) {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" %}} We'd love to accept your contributions to this project.
